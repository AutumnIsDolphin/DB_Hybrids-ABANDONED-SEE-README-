Requires [MelonLoader](https://melonloader.co/)

Mod Install:
1. Download latest release from [Releases]((https://github.com/AutumnIsDolphin/DB_Hybrids/releases)).
2. Put the `DB_Hybrids.dll` in your mods folder.
3. Launch the game.

Art Install:
1. Download latest release of [ArtSwap](https://github.com/ITR13/Demon-Bluff-Art-Swap-Mod).
2. Launch the game to let the folders appear.
3. Download the `art_cute.png` files.
4. Navigate to your mods folder where you installed the .dll (usually C:\ProgramFiles(x86)\Steam\steamapps\common\Demon Bluff Playtest\Mods\).
5. Enter the ArtSwap Folder.
6. Navigate to the folders for `Hout` and `Scover.`
7. Put the corresponding art_cute.png file in the folder and rename it to `art_cute`
